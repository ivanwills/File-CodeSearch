=head1 File::CodeSearch

The C<cs> command allows easy searching of files for text

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install

=head1 SUPPORT AND DOCUMENTATION

After installing, you can find documentation for this module with the
perldoc command.

    cs --help
    perldoc File::CodeSearch
